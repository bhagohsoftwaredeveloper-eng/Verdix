# **App Name**: StockPilot

## Core Features:

- Product Catalog: Manage product information, including name, description, price, and quantity.
- Sales Transactions: Record sales transactions, including items sold, quantities, and payment methods.
- Inventory Tracking: Monitor inventory levels and receive alerts when stock is low.
- Reporting & Analytics: Generate reports on sales, inventory, and other key metrics.
- AI-Powered Restock Suggestions: Uses sales trends to suggest optimal restock quantities for each product, ensuring sufficient inventory levels while minimizing storage costs; this feature acts as a tool to inform decisions about which restock recommendations to implement.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for trustworthiness and stability.
- Background color: Very light blue (#F0F2F7) to provide a calm and professional backdrop.
- Accent color: Purple (#7E57C2) to create visual interest and highlight key actions.
- Body and headline font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality.
- Use clear, minimalist icons to represent product categories, actions, and alerts.
- Design a clean and intuitive layout with easy-to-navigate menus and clear data presentation.
- Incorporate subtle transitions and animations to enhance user engagement and provide feedback on actions.
